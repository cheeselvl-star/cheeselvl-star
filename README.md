## What I'm building

**[crypto-grid-bot](https://github.com/cheeselvl-star/crypto-grid-bot)**
Neutral grid trading bot for crypto perpetual futures. Runs 24/7 as a systemd service on a
Raspberry Pi. Config-driven, multi-exchange via ccxt, includes regime kill switches (ADX +
200-day MA) that pause trading during trending markets. Python, asyncio, SQLite.

**[social-autopilot](https://github.com/cheeselvl-star/social-autopilot)**
CLI toolkit for automating video publishing to YouTube and Facebook Pages. OAuth token
auto-refresh, resumable uploads, cron-ready. Real production code running daily.
Python, Google API, Facebook Graph API.

**[review-reply-ai](https://github.com/cheeselvl-star/review-reply-ai)**
AI-powered Google review response generator for small businesses. Supports English,
Bahasa Malaysia, Mandarin, Tamil, and bilingual responses. Tone auto-adjusts by star
rating. Next.js 16, React 19, TypeScript, DeepSeek API.
